# AstroFinSentinelV5 — Success Criteria

**ATOM-META-RL-024 | Phase 7: Success Metrics Framework**

---

## Primary Trading Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| Sharpe Ratio | > 1.5 | Annualized, risk-free rate adjusted |
| Win Rate | > 55% | Profitable trades / total trades |
| Max Drawdown | < 15% | Peak-to-trough equity decline |
| Profit Factor | > 1.5 | Gross profit / gross loss |
| Expectancy | > 0.5% | Average return per trade |

## Risk-Adjusted Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| Sortino Ratio | > 2.0 | Return / downside deviation |
| Calmar Ratio | > 1.0 | Annual return / max drawdown |
| Recovery Factor | > 3.0 | Net profit / max drawdown |

## Operational Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| System Uptime | > 99.5% | Hours operational / total hours |
| Signal Latency | < 500ms | Time from query to signal |
| Agent Selection Diversity | > 3 agents/session | Unique agents selected per session |
| Belief Convergence | < 50 sessions | Sessions until belief stabilises |

---

Metrics are reviewed weekly during active trading and monthly during development phases.
